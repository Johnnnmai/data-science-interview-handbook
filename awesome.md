# Awesome Data Science Interview Resources

A curated list of resources for preparing for data science interviews. Organized by category.

Credit: [Alexey Grigorev](https://github.com/alexeygrigorev) for the original resource compilation format.


## Books

- [The 0→1 Data Scientist Interview Playbook](https://valenx.org/books/ds) - comprehensive DS interview preparation system based on 1,950+ job descriptions and 90+ real interview experiences
- [Ace the Data Science Interview](https://www.acethedatascienceinterview.com/) - 201 practice questions covering SQL, statistics, ML, product sense, and case studies
- [An Introduction to Statistical Learning](https://www.statlearning.com/) - free textbook covering core ML methods with intuition and R/Python code
- [Practical Statistics for Data Scientists](https://www.oreilly.com/library/view/practical-statistics-for/9781492072935/) - statistical concepts with practical examples in Python and R
- [Naked Statistics](https://wwnorton.com/books/Naked-Statistics/) - accessible introduction to statistical thinking for non-statisticians
- [The Elements of Statistical Learning](https://hastie.su.domains/ElemStatLearn/) - free advanced reference for ML and statistics
- [Causal Inference: The Mixtape](https://mixtape.scunning.com/) - free textbook on causal inference methods
- [Trustworthy Online Controlled Experiments](https://experimentguide.com/) - the definitive guide to A/B testing by practitioners from Microsoft
- [Designing Machine Learning Systems](https://www.oreilly.com/library/view/designing-machine-learning/9781098107956/) - ML system design for production
- [Python for Data Analysis](https://wesmckinney.com/book/) - pandas, NumPy, and data manipulation by the pandas creator


## Online Courses

- [Andrew Ng - Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction) - the most popular ML course, covers fundamentals clearly
- [StatQuest with Josh Starmer](https://www.youtube.com/c/joshstarmer) - statistics and ML concepts explained visually
- [fast.ai Practical Deep Learning](https://course.fast.ai/) - top-down approach to deep learning, free
- [Khan Academy Statistics](https://www.khanacademy.org/math/statistics-probability) - free fundamentals of statistics and probability
- [MIT 18.05 Introduction to Probability and Statistics](https://ocw.mit.edu/courses/18-05-introduction-to-probability-and-statistics-spring-2022/) - rigorous probability and statistics, free
- [DataTalks.Club ML Zoomcamp](https://mlzoomcamp.com/) - free practical ML course with projects
- [Brilliant.org Statistics](https://brilliant.org/courses/statistics/) - interactive statistics learning
- [Mode SQL Tutorial](https://mode.com/sql-tutorial/) - practical SQL for data analysis
- [Stanford CS229 Machine Learning](https://cs229.stanford.edu/) - graduate-level ML, free lecture notes and videos


## Practice Platforms

- [StrataScratch](https://www.stratascratch.com/) - SQL and Python interview questions from real companies (Meta, Amazon, Google)
- [LeetCode Database](https://leetcode.com/problemset/database/) - SQL problems ranked by difficulty
- [HackerRank SQL](https://www.hackerrank.com/domains/sql) - SQL practice with immediate feedback
- [DataLemur](https://datalemur.com/) - SQL interview questions from FAANG companies
- [Kaggle](https://www.kaggle.com/) - datasets, competitions, and community notebooks

- [SQLZoo](https://sqlzoo.net/) - interactive SQL tutorials and exercises
- [W3Schools SQL](https://www.w3schools.com/sql/) - SQL reference and practice

- [Netflix Tech Blog - Experimentation](https://netflixtechblog.com/tagged/experimentation) - how Netflix runs experiments at scale
- [Uber Engineering - Experimentation](https://www.uber.com/blog/tag/experimentation/) - experimentation platform and methodology

- [Causal Inference for the Brave and True](https://matheusfacure.github.io/python-causality-handbook/) - free online book on causal inference with Python
